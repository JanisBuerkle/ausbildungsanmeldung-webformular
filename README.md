# Ausbildungsanmeldung – Webformular

Dieses Projekt ist im Rahmen einer schulischen Gruppenarbeit im Bereich Webentwicklung entstanden.
Ziel war die Entwicklung eines vollständigen webbasierten Anmeldeformulars mit Frontend- und Backend-Anbindung.

Mein persönlicher Fokus lag auf dem **Frontend (HTML, CSS, JavaScript)** – insbesondere auf Struktur, Layout und clientseitiger Validierung.

---

## Projektübersicht

Das Projekt stellt ein webbasiertes Anmeldeformular für Ausbildungsplätze dar.
Nutzer können ihre Daten über ein übersichtlich aufgebautes Formular eingeben, welche anschließend geprüft und serverseitig weiterverarbeitet werden.

Der Schwerpunkt lag auf sauberer Struktur, guter Benutzerführung und einer klaren Trennung zwischen Frontend und Backend.

---

## Technologien

### Frontend (mein Verantwortungsbereich)
- HTML5 – semantischer und strukturierter Seitenaufbau
- CSS3 – Layout, Gestaltung und einfache Responsivität
- JavaScript – clientseitige Validierung und Nutzerfeedback

### Backend (Teamarbeit)
- PHP – Verarbeitung der Formulardaten
- MySQL / SQL – Datenbankstruktur und Datenspeicherung

---

## Frontend-Funktionen

- Übersichtliche und logisch aufgebaute Formularfelder
- Pflichtfeld-Prüfung auf Client-Seite
- Grundlegende Validierung typischer Eingaben
- Visuelles Feedback bei fehlerhaften oder fehlenden Eingaben
- Saubere Trennung von Struktur (HTML), Design (CSS) und Logik (JavaScript)

Das Formular ist auch ohne JavaScript grundsätzlich nutzbar und wird durch JavaScript funktional erweitert.

---

## Projektstruktur

```text
/
├── index.html        # Startseite mit Anmeldeformular
├── style.css         # Styling und Layout
├── main.js           # Clientseitige Validierung
├── README.md         # Projektdokumentation
└── Model/
    ├── anmeldung.php     # Serverseitige Formularverarbeitung
    ├── createdb.sql     # SQL-Skript zur Erstellung der Datenbank
    └── db_reqs.txt      # Hinweise und Anforderungen zur Datenbank
```

## Zusammenarbeit

Das Projekt wurde von drei Personen gemeinsam umgesetzt.

- **Mein Beitrag**
  - Frontend-Entwicklung mit HTML, CSS und JavaScript
  - Aufbau und Strukturierung des Formulars
  - Gestaltung des Layouts und der Benutzeroberfläche
  - Umsetzung der clientseitigen Validierung und Nutzerführung

- **Beiträge der Teammitglieder**
  - Implementierung der serverseitigen Logik mit PHP
  - Planung, Erstellung und Anbindung der SQL-Datenbank

Die Aufgabenverteilung war klar definiert, sodass jedes Teammitglied eigenständig an seinem Bereich arbeiten konnte. Die Zusammenarbeit erfolgte abgestimmt und zielorientiert.
